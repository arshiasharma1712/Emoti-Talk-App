# Emoti-Talk App

## Overview

This repository contains a simple web application designed to help special-needs children communicate emotions and basic needs using emoticons and speech synthesis.

The goal of the project was to explore how a visual interface combined with audio feedback could support children who may have difficulty expressing their feelings verbally.

Users can click on emoticons representing emotions or actions, and the application will display and speak the corresponding message.

---

## What I Built

The application provides a simple interface where children can communicate by selecting icons representing:

- emotions such as happy, sad, sick, sleepy, angry, and scared  
- needs such as eating, playing, or drinking water  

When an icon is clicked, the application:

- displays a text message
- speaks the message using browser speech synthesis
- briefly shows the message above the icon

## Accessibility Focus

This project was designed with accessibility in mind:

- large clickable icons
- simple visual layout
- text feedback
- audio speech output

These features help support children who rely on visual or auditory communication aids.

Working on this project helped me explore:

- building interactive web interfaces
- designing simple accessible UI for communication

- combining visual elements with audio feedback
